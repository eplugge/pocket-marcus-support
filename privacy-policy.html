1. INTRODUCTION

Pocket Marcus ("we," "our," or "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, and protect your information when you use our mobile application ("App") that provides AI-powered Stoic philosophy guidance.

2. INFORMATION WE COLLECT

2.1 Information You Provide

  • Conversation Messages: Text messages you send to interact with Marcus Aurelius AI
  • API Keys: If using your own API keys, they are stored locally on your device
  • Subscription Information: Purchase and subscription data through App Store

2.2 Automatically Collected Information

  • Usage Data: App interaction patterns, response preferences, and feature usage
  • Device Information: Device type, operating system version, app version

2.3 Information We Do NOT Collect

  • Personal identifying information (name, email, phone number)
  • Location data
  • Device contacts or photos
  • Biometric data

3. THIRD-PARTY AI SERVICES

3.1 AI Provider Integration

Our App integrates with multiple AI services to provide responses:

Google Gemini API
  • Data Sent: Your conversation messages and context
  • Purpose: Generate philosophical responses
  • Privacy Policy: https://policies.google.com/privacy
  • Data Processing: Google processes your messages according to their API terms
OpenAI API
  • Data Sent: Your conversation messages and context
  • Purpose: Generate philosophical responses
  • Privacy Policy: https://openai.com/privacy
  • Data Processing: OpenAI processes your messages according to their API terms
Anthropic Claude API
  • Data Sent: Your conversation messages and context
  • Purpose: Generate philosophical responses
  • Privacy Policy: https://www.anthropic.com/privacy
  • Data Processing: Anthropic processes your messages according to their API terms
Ollama (Self-Hosted AI)
  • Data Processing: Sent to your configured Ollama server
  • Privacy: Data sent to your own server or network
  • Purpose: Private AI responses using your own infrastructure
  • Note: You control the server location and data handling

3.2 API Key Security

  • User API Keys: Your API keys are stored locally on your device using iOS Keychain
  • Other Tiers: We use our own API keys with built-in usage limits
  • Encryption: All API keys are encrypted and never transmitted to our servers

4. HOW WE USE YOUR INFORMATION

4.1 Primary Uses

  • Provide AI-powered philosophical guidance
  • Maintain conversation context for coherent responses
  • Track usage limits per subscription tier
  • Improve app performance and user experience

4.2 AI Processing

  • Your messages are sent to selected AI providers to generate responses
  • We do not store conversation history on our servers
  • AI providers may temporarily process your data according to their policies
  • Local AI (Ollama) processing never leaves your device

5. DATA STORAGE AND SECURITY

5.1 Local Storage

  • Conversation History: Stored locally on your device
  • User Preferences: Stored locally using iOS secure storage
  • API Keys: Stored in iOS Keychain (when using your own keys)

5.2 Server Storage

  • Subscription Status: Processed through App Store Connect
  • Usage Analytics: Anonymized and aggregated only
  • No Personal Data: We do not store personal information on our servers

5.3 Security Measures

  • End-to-end encryption for API communications
  • Secure storage using iOS security frameworks
  • Regular security audits and updates
  • No data transmission to unauthorized third parties

6. YOUR RIGHTS AND CHOICES

6.1 Data Control

  • Clear Conversation History: Delete all local conversation data
  • AI Provider Selection: Choose which AI services to use
  • Subscription Management: Cancel or modify through App Store

6.2 Data Access

  • Local Data: Access all your data stored locally on your device
  • No Server Data: We do not store personal data on our servers to access

6.3 Data Deletion

  • App Deletion: Removes all local data from your device
  • Conversation Clear: Option to clear chat history in app settings

7. CHILDREN'S PRIVACY

Our App is rated 4+ and suitable for all ages. We do not knowingly collect personal information from children under 13. The App focuses on philosophical education and does not contain inappropriate content for minors.
